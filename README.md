# Agent Arthur Donnelley - Delta Green Soundboard

A web-based soundboard designed to enhance the **Music from a Darkened Room** Delta Green scenario with authentic phone call audio from Agent Arthur Donnelley.

## Overview

This soundboard provides game masters with an immersive audio toolkit for running the _Music from a Darkened Room_ scenario, featuring Agent Donnelley's voice guiding players through their investigation of the haunted house at 1206 Spooner Avenue.

## Volume Controls

- **Static Volume** - Control the background phone static independently (default: 15%)
- **Voice Volume** - Adjust Donnelley's voice clips (default: 100%)

Both volumes can be adjusted in real-time during gameplay.

## Usage for Game Masters

### Setup

1. Open `index.html` in any modern web browser
2. Adjust volume levels to your preference
3. Optionally enable background static for atmosphere

### During Play

- Click any sound button to play Donnelley's voice
- Use the categorized sections to quickly find appropriate responses
- Layer static background for phone call immersion
- Adjust volumes on-the-fly as scenes develop

### Running the Scenario

The soundboard is organized to follow the natural flow of _Music from a Darkened Room_:

1. **Initial Contact** - Use introductions when establishing the phone call
2. **House Investigation** - Guide players room-by-room with exploration prompts
3. **Building Tension** - Layer in tension clips as the supernatural reveals itself
4. **Investigation** - Prompt players to describe details with investigation questions
5. **Managing Fear** - Use calming responses when players panic
6. **Delta Green Elements** - Introduce conspiracy and trust issues with Delta Green clips

### Tips

- The static effect adds authenticity to the "phone call" experience
- Stop currently playing clips by clicking another button
- Mix categories to create dynamic, realistic conversation flow
- Use the "Delta Green" section sparingly for maximum impact

## Technical Details

- Pure HTML/CSS/JavaScript - no dependencies required
- Works offline once downloaded
- Responsive design for tablets and mobile devices
- All audio files stored in `/sounds` directory
